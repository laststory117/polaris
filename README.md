## polaris - 北极星
可视化搭建工具，内部组件以jl命名

# 开发相关：app
## 安装依赖
```
yarn / npm install
```
## 开发模式

```
yarn dev / npm run dev / npm run serve
```
## 生产模式

```
yarn build / npm run build
```

# 开发相关：server
## 安装依赖
```
yarn / npm install
```
## 后台启动
```
yarn serve / node index.js
nodemon index.js  // 后台热更新启动
```
> 端口为localhost:3001
